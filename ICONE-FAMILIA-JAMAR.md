## 🛠️ **Solução de Problemas**

### **Ícone não abre:**
```bash
# Verificar Node.js
node --version

# Reinstalar dependências
npm install

# Tentar novamente
Família Jamar.vbs
```

### **Porta ocupada:**
```bash
# Matar processos
taskkill /F /IM node.exe

# Tentar novamente
Família Jamar.vbs
```

### **Navegador não abre:**
- **Verifique** se o antivírus não bloqueou
- **Execute** como administrador
- **Verifique** se a porta 3000 está livre
